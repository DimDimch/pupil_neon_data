# Эксперимент с айтрекером Pupil Neon — Инструкция по развёртыванию

## Что это

Программа для проведения когнитивного эксперимента (GAP, Overlap, Prediction, Decision, Antisaccade) с мониторингом движения глаз через Pupil Neon 200. Все данные айтрекера сохраняются **на компьютере** в реальном времени.

## Требования

- **Python 3.11** или новее (скачать с [python.org](https://www.python.org/downloads/))
- **macOS** (протестировано на macOS, но работает и на Windows)
- Компьютер и телефон с Companion App должны быть **в одной Wi-Fi сети**
- OBS Studio (опционально, для записи экрана)

## Установка (пошагово)

### 1. Установите Python

Если Python ещё не установлен, скачайте и установите с [python.org](https://www.python.org/downloads/). При установке поставьте галочку **"Add Python to PATH"**.

### 2. Откройте Терминал

- На Mac: Finder → Программы → Утилиты → Терминал
- На Windows: нажмите Win+R, напишите `cmd`, нажмите Enter

### 3. Перейдите в папку с программой

```bash
cd путь/к/папке/с/программой
```

Например:
```bash
cd ~/Desktop/experiment
```

### 4. Установите зависимости

```bash
pip install -r requirements.txt
```

Если команда не работает, попробуйте:
```bash
pip3 install -r requirements.txt
```

## Запуск эксперимента

```bash
python experiment.py
```

Или:
```bash
python3 experiment.py
```

### Что произойдёт:

1. Откроется **окно настройки** — заполните поля (ID испытуемого, группу, возраст и т.д.)
2. Нажмите **"Начать эксперимент"**
3. Откроется полноэкранное окно pygame с маркерами — сделайте скриншот (Shift+Cmd+3), нажмите **ПРОБЕЛ**
4. Программа подключится к Pupil Neon по указанному IP
5. Начнётся запись — данные стримятся на компьютер в реальном времени
6. После завершения всех блоков запись останавливается автоматически

## Выходные данные

После эксперимента в папке `data/` появится подпапка с данными:

```
data/{ID}_{Группа}_S{Сессия}_{дата_время}/
├── stimulus_log.csv   ← лог стимулов (как раньше EXP_*.csv)
├── events.csv         ← все отправленные маркеры
├── gaze.csv           ← сырые координаты взгляда (x, y, worn)
├── fixations.csv      ← события фиксации
├── blinks.csv         ← события моргания
├── imu.csv            ← акселерометр/гироскоп
└── eye_camera.mp4     ← видео камер глаз
```

### Описание файлов

- **stimulus_log.csv** — полностью совместим с прежним форматом EXP_*.csv. Содержит метаданные испытуемого и все события стимулов.
- **events.csv** — хронологический список всех маркеров, отправленных на устройство (timestamp_ns, tag).
- **gaze.csv** — координаты взгляда ~200 Гц. Поля: timestamp_s, x_norm, y_norm, worn (1 = очки надеты).
- **fixations.csv** — предрассчитанные фиксации с координатами и длительностью.
- **blinks.csv** — события моргания (начало, конец, длительность).
- **imu.csv** — данные акселерометра и гироскопа (если поддерживается прошивкой).
- **eye_camera.mp4** — видео камер глаз (если установлен imageio).

## Настройки сохраняются

Программа запоминает последние введённые значения в файле `last_session.json`. При следующем запуске поля формы будут предзаполнены.

## Частые проблемы

### "Не удалось подключиться к IP"

- Убедитесь, что телефон и компьютер в **одной Wi-Fi сети**
- Проверьте IP в Companion App (Настройки → Streaming)
- Попробуйте пингануть устройство: `ping 192.168.1.101`

### "ModuleNotFoundError: No module named 'pygame'"

Зависимости не установлены. Выполните:
```bash
pip install -r requirements.txt
```

### "eye_camera.mp4 не создан"

Для записи видео камер глаз нужен пакет `imageio[ffmpeg]`. Установите:
```bash
pip install "imageio[ffmpeg]"
```

### Экран чёрный, ничего не видно

Эксперимент работает в полноэкранном режиме. Нажмите **Escape** для выхода.

### OBS не записывает

- Убедитесь, что OBS запущен
- В OBS: Инструменты → Настройки WebSocket → включите сервер, порт 4455
- Пароль в программе: `XHSKSGDHSFYSLBSt` (можно изменить в experiment.py, переменная `OBS_PASSWORD`)

### Маркеры не отображаются

Убедитесь, что папка с маркерами существует:
- AprilTags (0-8): `/Users/iliachapliev/Desktop/маркеры/`
- Маркеры рамки (24-33): `~/Desktop/МАРКЕРЫ/`

## Изменение параметров эксперимента

Тайминги блоков можно менять в `experiment.py` — секция "Тайминги блоков":

```python
FIXATION_DURATION = 2.8
GAP_TIME = 0.2
TARGET_DURATION = 1.5
ITI = 0.5
# ... и т.д.
```

Количество триалов задаётся в функции `main()`:
```python
await blk(writer, n_trials=20)  # измените 20 на нужное число
```
